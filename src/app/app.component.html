What is this? This is a browser based MIDI Program Change 'scratch pad' for your whole studio. It will display program change data for all available MIDI inputs in real time, allow you store a snapshot, then restore it. When restore the snapshot, it will be sent to all available MIDI outputs.

It uses WebMidi.js and is running 100% in your browser. You can read the code on GitHub if you like.

Please report bugs here: https://github.com/Skeyelab/MIDI-Program-Master/issues
