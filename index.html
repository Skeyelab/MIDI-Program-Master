MIDI Program Master

What is this? This is a browser based MIDI Program Change 'scratch pad' for your whole studio.
It should display program change data for all available MIDI inputs in real time, allow you store a snapshot, then recall it.

It uses WebMidi.js and is running 100% in your browser. You can read the code on GitHub if you like.
